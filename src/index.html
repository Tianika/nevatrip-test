1.2 Таблица на странице

Горизонтальный скролл появляется из-за того, что на странице есть вертикальный скролл. Самый простой способ - это использовать overflow-x: hidden для body. Более правильно немного поменять стиль для div с классом blog 
                .blog { 
                  position: relative; 
                  margin: 0 auto; 
                  padding: 20px 15px; 
                  background-color: #fff; 
                  box-sizing: border-box; 
                  max-width: 990px;
                  width: 100%; 
                }
